# CLAUDE.md

This repo contains instructions for building the reader's own **HQ**, a file-based memory system and companion for a coding agent.

If the user asks you to set up their HQ (for example, "read AGENTS.md and set up my HQ"), read **[AGENTS.md](AGENTS.md)** and follow it start to finish. That file is the complete bootstrap procedure.

Note: this repo is the *instructions*, not the HQ itself. You are building the user's HQ somewhere else, `~/hq` by default, per AGENTS.md. Don't turn this repo into their HQ.
